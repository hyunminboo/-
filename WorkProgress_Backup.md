# 로비 UI 및 캐릭터 선택 시스템 작업 내역 백업 (2026-06-09)

내일 이어서 작업하실 수 있도록 오늘 진행된 주요 수정 사항과 현재 상태를 정리해 둔 문서입니다.

## 1. 로비 화면 디자인 전면 개편 (SF/메카닉 테마)
* **배경 이미지 교체:** 딱딱했던 단색 배경 대신, 고해상도의 사이버틱한 지휘 통제실(Hangar) 2D 배경(`Lobby_SciFi_Background.png`)을 적용했습니다.
* **글래스모피즘(Glassmorphism) UI 적용:**
  * 기존의 불투명한 회색 패널들을 모두 **반투명한 검은색 유리 느낌**(`Alpha 0.7`)으로 교체하여 배경이 은은하게 비치도록 만들었습니다.
  * 텍스트 색상을 **네온 시안(Cyan) 및 네온 그린(Green)** 계열로 통일하여 홀로그램 인터페이스 느낌을 강조했습니다.
* **둥근 모서리(Rounded Corners) 패널 적용:**
  * 딱딱하게 각져있던 UI 패널과 버튼들의 모서리를 둥글게 처리했습니다.
  * 유니티의 9-Slicing 기법을 적용해 크기가 변해도 모서리가 찌그러지지 않도록 세팅했습니다.

## 2. 난이도 선택 피드백 시각화
* `DifficultySelector.cs` 스크립트를 수정하여 난이도 버튼 클릭 시 시각적 피드백을 강화했습니다.
* **선택 전:** 어두운 유리 느낌
* **선택 후:** 테두리 색상과 패널 밝기가 난이도별로 확연히 달라짐
  * **NORMAL:** 초록색 네온
  * **HARD:** 주황색 네온
  * **NIGHTMARE:** 빨간색 네온
* (버그 수정) 난이도 버튼 뒤쪽에 검은색 찌꺼기 텍스처나 하늘색 잔상이 남던 문제를 해결했습니다.

## 3. 캐릭터 선택 (AGENT SELECT) 시스템 추가
* 우측 패널(RightPanel_Info) 난이도 선택창 바로 위에 캐릭터를 고를 수 있는 **[SELECT AGENT]** 영역을 신규 추가했습니다.
* **버튼 크기 및 비율 완벽 통일:**
  * 남캐(`Player_Male.png`)와 여캐(`player.png`) 스프라이트의 여백 차이로 인해 발생하던 크기/비율 왜곡 현상을 수정했습니다.
  * 여캐 아이콘의 Scale을 1.8배로 키우고 가로세로 비율(preserveAspect)을 잠가서, 두 캐릭터가 눈으로 볼 때 똑같은 덩치로 보이게 맞췄습니다.
* **LobbyManager.cs 연동 완료:**
  * `CharacterSelector.cs` 스크립트를 새로 작성하여 선택된 캐릭터의 데이터를 관리합니다.
  * 로비에서 **[START OPERATION]** (작전 시작) 버튼을 누르면, **게임에 배치된 Player의 Sprite(외형)가 즉시 선택한 캐릭터로 교체**되도록 연동했습니다.
  * 캐릭터 외형 변경 시, 총알이 스쳐 지나가거나 억울하게 맞는 일이 없도록 **PolygonCollider2D(피격 판정 히트박스)도 새 외형에 맞춰 자동으로 재계산** 되도록 처리했습니다.

## 4. 내일 이어서 진행할 수 있는 추가 작업 제안
1. **인게임 무기 장착 보정:** 새로 추가된 남캐의 경우 손 위치가 여캐와 미세하게 다를 수 있으므로, 총기를 들었을 때 총의 위치가 자연스러운지 테스트 및 미세 조정이 필요할 수 있습니다.
2. **캐릭터별 애니메이션/스탯 차별화:** 현재는 외형(Sprite)만 바뀌고 있으나, 차후 남캐/여캐별로 전용 애니메이션(Animator Controller)이나 이동 속도, 체력 등의 고유 스탯을 부여하는 작업을 진행할 수 있습니다.
3. **무기 선택 시스템 연동:** 캐릭터 선택창처럼, 시작할 때 들고 갈 기본 무기를 선택하는 로비 UI도 추가로 구성할 수 있습니다.
